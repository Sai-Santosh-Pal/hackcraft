HACKCRAFT

A Minecraft Look-Alike
Made In Python & Ursina Engine

Download Instructions

  • Unzip the downloaded .zip file
  • Open the folder - 'Minecraft' - in your command prompt
  • Make sure you have Python installed
  • Run by typing - python main.py

How To Play?

  • WASD to move, Spacebar to jump
  • Right click to break, Left click to place
  • 1-4 to change selected item from hotbar
  • If you're stuck in a block press Spacebar
  • Press Q to quit
